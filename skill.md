# 내재가치 계산 스킬 (한국 주식 퀀트 분석)

## 핵심 공식

```
가중EPS = 최근년도EPS × 3 + 직전년도EPS × 2 + 전전년도EPS × 1
내재가치 = (최근BPS + 가중EPS) / 2
```

---

## 사전 처리 규칙

### 1. (E) 데이터 제외

컬럼명에 `(E)` 표기가 있는 추정치는 반드시 제외한다.

```
예시 제외 대상:
  2025/12(E) (IFRS연결)
  2026/03(E) (IFRS연결)
```

### 2. 결산월 확인

- **12월 결산**: 연간 기준 컬럼이 YYYY/12 형태
- **3월 결산**: 연간 기준 컬럼이 YYYY/03 형태 (일본계, 해운, 일부 유통업 등)
- 결산월에 따라 "가장 최근 연도"의 기준이 달라짐

### 3. 사용 EPS/BPS 확인

- 원칙적으로 공시 데이터의 EPS, BPS를 그대로 사용
- EPS는 **지배주주 귀속 기준** 우선 사용 (표에 별도 구분이 있는 경우)
- BPS도 **지배주주 기준** 우선 사용

---

## 방법 1: 연간 데이터만 있을 때

### 데이터 추출

| 항목         | 기준                   |
| ------------ | ---------------------- |
| BPS          | 가장 최근 결산연도 BPS |
| 최근년도 EPS | 가장 최근 결산연도 EPS |
| 직전년도 EPS | 최근-1년 EPS           |
| 전전년도 EPS | 최근-2년 EPS           |

### 계산 순서

**Step 1. 가중EPS 계산**

```
가중EPS = EPS(n) × 3 + EPS(n-1) × 2 + EPS(n-2) × 1
```

**Step 2. 내재가치 계산**

```
내재가치 = (BPS(n) + 가중EPS) / 2
```

### 예시 (12월 결산, 2024년이 최근년도)

```
n   = 2024  →  EPS(2024) = 4,950원,  BPS = 57,981원
n-1 = 2023  →  EPS(2023) = 2,131원
n-2 = 2022  →  EPS(2022) = 8,057원

가중EPS = 4,950×3 + 2,131×2 + 8,057×1 = 27,169원
내재가치 = (57,981 + 27,169) / 2 = 42,575원
```

---

## 방법 2: 연간 + 분기 데이터가 함께 있을 때

분기 데이터가 있으면 "가장 최근년도"를 **직전 4분기 합산 추정치**로 대체한다.

### 데이터 추출

| 항목         | 기준                             |
| ------------ | -------------------------------- |
| BPS          | **가장 최근 분기말** BPS         |
| 최근년도 EPS | **직전 4분기 EPS 합산**          |
| 직전년도 EPS | 직전 연간 EPS (E 제외 실적 기준) |
| 전전년도 EPS | 전전 연간 EPS (E 제외 실적 기준) |

### 직전 4분기 선택 기준

가장 최근 분기가 Q라면:

```
직전 4분기 = Q-3, Q-2, Q-1, Q
```

예시: 마지막 분기가 2025/09이면

```
4분기 = 2024/12 + 2025/03 + 2025/06 + 2025/09
```

### 계산 순서

**Step 1. 추정 연간EPS 계산**

```
추정연간EPS = EPS(Q-3) + EPS(Q-2) + EPS(Q-1) + EPS(Q)
```

**Step 2. 가중EPS 계산**

```
가중EPS = (연간EPS × 3 + EPS(전년연간) × 2 + EPS(전전년연간) × 1) / 6 * 10
```

**Step 3. 내재가치 계산**

```
내재가치 = (BPS(최근분기말) + 가중EPS) / 2
```

### 예시 (12월 결산, 마지막 분기 2025/09)

```
BPS = BPS(2025/09) = 60,632원

추정연간EPS = EPS(2024/12) + EPS(2025/03) + EPS(2025/06) + EPS(2025/09)
            = 1,115 + 1,186 + 733 + 1,783 = 4,817원

직전년도 = 2024 연간 EPS = 4,950원
전전년도 = 2023 연간 EPS = 2,131원

가중EPS = 4,817×3 + 4,950×2 + 2,131×1 = 26,482원
내재가치 = (60,632 + 26,482) / 2 = 43,557원
```

## 결과 출력 형식

계산 후 아래 형식으로 출력한다.

### 계산 테이블

| 항목          | 방법 1 (연간)     | 방법 2 (연간+분기) | 변화       |
| ------------- | ----------------- | ------------------ | ---------- |
| BPS           | XXX원             | XXX원              | ▲/▼ X%     |
| 추정 연간 EPS | XXX원             | XXX원              | ▲/▼ X%     |
| 가중 EPS      | XXX원             | XXX원              | ▲/▼ X%     |
| **내재가치**  | **XXX원**         | **XXX원**          | **▲/▼ X%** |
| 추정 현재주가 | XXX원 (PBR X.XX×) | XXX원 (PBR X.XX×)  | —          |
| 저평가 여부   | 저/고평가 (X%)    | 저/고평가 (X%)     | —          |

### 추정 현재주가 계산

```
추정 현재주가 = 가장 최근 BPS × 가장 최근 PBR
```

### 괴리율 계산

```
괴리율(%) = (현재주가 - 내재가치) / 내재가치 × 100
  → 음수(-): 저평가
  → 양수(+): 고평가
```

## 주의사항 및 체크리스트

계산 완료 후 아래 항목을 검토하고, 해당 사항이 있으면 특이사항으로 언급한다.

### ⚠️ 반드시 확인

- [ ] (E) 추정 데이터가 섞여 있지 않은가
- [ ] 3월 결산인지 12월 결산인지 확인했는가
- [ ] 분기 EPS 중 이상치(일회성 손익)가 있는가
  - 특정 분기 EPS가 전후 분기 대비 3배 이상이면 언급
- [ ] EPS가 음수(적자)인 연도가 있는가
  - 가중EPS를 왜곡할 수 있으므로 주의

### 💡 추가 맥락 제공

- 부채비율이 200% 초과 시: 금융업 또는 고레버리지 업종 가능성 언급
- PBR이 1배 미만: 자산가치 대비 저평가 언급
- ROE가 5% 미만: 자본 효율성 낮음 언급
- 최근 분기에서 주가 급등(PBR 급상승)이 있으면 언급

## 한계 및 면책

이 스킬은 단순 공식 기반의 참고용 분석이다.

- 실제 투자 결정은 추가적인 정성 분석과 전문가 조언을 병행해야 한다.
- 내재가치 공식은 절대적 기준이 아니며, 업종/비즈니스 특성에 따라 해석이 달라질 수 있다.
